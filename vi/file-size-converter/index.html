How to use

  1. Type the number you want to convert.
  2. Pick the source and target units.
  3. Pick the unit system. Drive manufacturers and most networking specs use decimal (1 KB = 1000 B). Operating systems (Windows, Linux, macOS in older versions) and RAM specs use binary (1 KiB = 1024 B).

The naming convention KiB / MiB / GiB exists specifically to disambiguate binary from decimal. "KB" without the i usually means decimal but is sometimes used loosely. When in doubt, decimal is the formal SI standard.
