How to use

  1. The current Unix time updates live at the top — click Use Now to copy it into the converter.
  2. To convert a timestamp to a date: paste the number and press Convert. Seconds (10 digits) and milliseconds (13 digits) are auto-detected.
  3. To convert a date string to a timestamp: paste an ISO 8601 string (2024-01-15T12:00:00Z) or just 2024-01-15 and press Convert.
  4. Both UTC and your local time-zone are shown so you don't need to do time-zone math.

Unix time counts seconds since 1970-01-01 UTC, ignoring leap seconds. The 32-bit signed range overflows in 2038 — most modern systems use 64-bit and are unaffected.

Examples

Timestamp → date:

1700000000      → 2023-11-14T22:13:20Z (UTC)
1700000000000   → 2023-11-14T22:13:20.000Z
0               → 1970-01-01T00:00:00Z (Unix epoch)

Date → timestamp:

2024-01-15T12:00:00Z → 1705320000  (seconds)
                     → 1705320000000 (ms)
