🎯 Few-Shotプロンプトフォーマッター

🔒 Secure & Client-side

システム指示と入力→出力の例リストを追加すると、Markdown・Anthropicスタイルのタグ・OpenAI messages JSON・プレーンテキスト形式のFew-shotプロンプトを生成します — ChatGPT・Claude・スクリプトにそのまま貼り付けられます。

フォーマットされたプロンプト


                

                    
                    
                

            


            
このツールはテキストを再フォーマットするだけで、AIプロバイダーには接続しません。有料APIに送信する前に結果を必ず確認してください。


            
使い方

            
  1. （任意）タスクやペルソナを定義するシステム指示を入力します。
  2. + 例を追加をクリックして入力/出力ペアを追加します。Few-shotには通常3〜5例が適切です。
  3. 出力形式を選択してください:
    • Markdown — 読みやすく、チャットUIに向いています。
    • XMLタグ — AnthropicはClaude向けに各ペアを<example>タグで囲むことを推奨しています。
    • OpenAI messages JSON — {role, content}オブジェクトの配列で、Chat Completions APIにそのまま渡せます。
    • プレーン — マークアップなしのシンプルな形式。
  4. フォーマットをクリックして結果をコピーします。
Claude/AnthropicにはXMLタグが例の構造遵守を一貫して改善します。ChatGPTにはMarkdownまたはmessages JSONが通常最適です。OpenAI形式は各ペアをuser/assistantターンで交互に配置し、APIが会話履歴として処理します。
