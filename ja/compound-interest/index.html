📈 複利計算機

🔒 Secure & Client-side

初期元金・年利・期間・複利頻度・月々の積立を入力すると、投資や貯蓄口座の将来価値を計算します。年別残高推移も表示されます。

推定値のみ — 金融アドバイスではありません。実際のリターンは変動し、保証されません。

使い方

  1. 現在持っている初期元金を入力します。
  2. 期待する年利率をパーセントで入力します（S&P 500の長期平均は名目約10%、実質約7%）。
  3. 投資期間を年単位で入力します。
  4. 複利頻度を選びます: 年1回・月・日。インデックスファンドは実質連続複利のため月複利が近似値です。
  5. 月々の積立額を入力（なければ0）して計算をクリックします。

計算式: FV = P(1 + r/m)m·t + C · ((1 + r/m)m·t − 1) / (r/m)。Pは元金、rは年利率、mは年間複利回数、tは年数、Cは定期積立額です。

インフレ・税金・手数料は考慮されていません。名目7%でインフレ3%であれば実質利回りは約4% — 将来残高の購買力は表示される数値より低くなります。
