💾 ファイルサイズ変換

🔒 Secure & Client-side

バイト・KB・MB・GB・TB・PB単位を自由に変換します。ハードドライブ広告で使われる10進法(1KB = 1000B)と、OSが表示する2進法(1KiB = 1024B)を選択できます。

結果
—

使い方

  1. 変換する数値を入力します。
  2. 変換元と変換先の単位を選びます。
  3. 単位体系を選択します。ハードドライブメーカーやネットワーク仕様は10進法(1KB = 1000B)、OS(Windows・Linux・旧macOS)やRAM仕様は2進法(1KiB = 1024B)を使います。

KiB・MiB・GiB表記は2進法であることを明確に区別するためのIEC標準です。"KB"は通常10進法ですが、緩い意味で2進法にも使われます — 正確な仕様には10進法がSI公式標準です。
