🔌 MCPサーバー設定ジェネレーター

🔒 Secure & Client-side

Claude Desktop・Cursor・VS Code向けのModel Context Protocol（MCP）サーバー設定を生成します。トランスポート・コマンド・引数・環境変数を入力すると、各クライアントに対応した正しいJSONと設定ファイルのパスを出力します。

対象クライアント:

設定JSON

インストールパス: 



            
使い方

            
  1. クライアント（Claude Desktop・Cursor・VS Code）を選びます。
  2. サーバー名（辞書のキーとして使用）を入力します。
  3. トランスポートを選択: stdioはローカルプロセス（最も一般的）、streamable-http・sseはリモートサーバー向け。
  4. stdioの場合: コマンドと引数を入力 / HTTP・SSEの場合: URLを入力。
  5. 必要に応じて環境変数を追加。
  6. 生成後、表示されたパスの設定ファイルにJSONをコピーします。
設定ファイルの場所:
Claude Desktop — macOS: ~/Library/Application Support/Claude/claude_desktop_config.json; Windows: %APPDATA%\Claude\claude_desktop_config.json
Cursor — グローバル: ~/.cursor/mcp.json; プロジェクト別: .cursor/mcp.json
VS Code — ワークスペース: .vscode/mcp.json (またはユーザー設定 → MCP Servers)
