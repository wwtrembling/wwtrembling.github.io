JSON 포매터

🔒 Secure & Client-side

JSON 데이터를 포맷팅하고 유효성을 검사하세요. 개발자를 위해 광고를 제거했습니다.

사용 방법

  1. 왼쪽 패널에 JSON 데이터를 붙여넣으세요.
  2. Format으로 2-space 들여쓰기 적용, Minify로 공백 제거, Validate로 문법만 검사할 수 있습니다.
  3. Copy 버튼으로 결과를 클립보드에 복사하세요.
  4. 모든 처리는 브라우저에서만 이루어지며 데이터가 서버로 전송되지 않습니다.

1 MB 이상 입력은 페이지가 잠깐 멈출 수 있어 확인 창이 표시됩니다. 15자리 초과 정수는 JavaScript의 64-bit float 한계로 정밀도 손실 경고가 표시됩니다.

예시

축약된 JSON을 보기 좋게 정렬:

입력:  {"name":"Ada","tags":["dev","math"]}
출력: {
  "name": "Ada",
  "tags": [
    "dev",
    "math"
  ]
}

후행 쉼표 오류 검출:

입력:  {"a": 1, "b": 2,}
결과: ✗ Invalid JSON — Unexpected token } in JSON
