사용 방법

            
  1. 프로젝트 이름과 역할 설명을 입력하세요.

  3. 제약 사항도 한 줄에 하나씩.
  4. 자주 쓰는 명령어(테스트·린트·개발 서버 등)를 입력 — ```bash 블록으로 감싸집니다.
  5. 생성을 눌러 검토 후 .md 다운로드로 저장하면 레포 루트의 CLAUDE.md로 사용할 수 있습니다.
같은 파일은 Cursor(.cursorrules), Aider 등 다른 에이전트의 시스템 프롬프트로도 사용 가능합니다. Cursor는 다운로드한 파일 이름만 바꾸면 됩니다.
