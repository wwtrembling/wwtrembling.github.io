⏲️ Unix 타임스탬프 변환기

🔒 Secure & Client-side

Unix 타임스탬프와 날짜를 변환합니다. 초/밀리초를 자동 감지하고 UTC와 로컬 시간을 함께 보여줍니다.

사용 방법

  1. 상단에 현재 Unix 시간이 실시간 표시됩니다 — Use Now로 변환기에 입력할 수 있습니다.
  2. 타임스탬프를 날짜로: 숫자를 붙여넣고 Convert를 누르세요. 초(10자리)와 밀리초(13자리)가 자동 감지됩니다.
  3. 날짜를 타임스탬프로: ISO 8601(2024-01-15T12:00:00Z) 또는 2024-01-15 형식을 입력하고 Convert를 누르세요.
  4. UTC와 로컬 타임존이 함께 표시되어 시간대 계산이 불필요합니다.

Unix time은 1970-01-01 UTC 이후의 초를 세며 윤초는 무시합니다. 32-bit signed는 2038년에 오버플로하지만 대부분의 현대 시스템은 64-bit이므로 영향이 없습니다.

예시

타임스탬프 → 날짜:

1700000000      → 2023-11-14T22:13:20Z (UTC)
1700000000000   → 2023-11-14T22:13:20.000Z
0               → 1970-01-01T00:00:00Z (Unix epoch)

날짜 → 타임스탬프:

2024-01-15T12:00:00Z → 1705320000  (초)
                     → 1705320000000 (밀리초)
