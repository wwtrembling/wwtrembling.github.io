ChatGPT·Claude·Gemini 등에 프롬프트를 붙여넣기 전 민감 정보(이메일·전화·주민번호·카드·API 키·IP·URL)를 자동 마스킹. 모든 처리는 브라우저에서 이루어집니다.

마스킹된 항목:

    사용 방법

    1. AI에 보낼 프롬프트 또는 문서를 붙여넣으세요.
    2. 마스킹할 카테고리를 선택합니다 (기본: 전부 활성).
    3. 마스킹을 누르면 오른쪽에 [EMAIL], [PHONE] 등의 자리표시자로 치환된 결과가 표시됩니다.
    4. 마스킹된 텍스트를 복사해 ChatGPT·Claude·Gemini 등에 붙여넣으세요.

    감지되는 항목: 일반 이메일; 국제 표준 전화번호; 한국 주민등록번호와 미국 SSN; Luhn 검증을 통과한 신용카드 번호; 잘 알려진 접두사(sk-, pk-, AKIA, AIza, github_pat_, xoxb-)의 API 키; IPv4 주소; HTTP(S) URL.

    한계: 자유 형식 주소, 이름, 독자 ID는 감지되지 않습니다. 보내기 전 결과를 반드시 확인하세요.
