🔐 Base64 인코더/디코더

🔒 Secure & Client-side

텍스트를 Base64로 인코딩하거나 디코딩하세요.

사용 방법

  1. 입력 칸에 텍스트를 입력하거나 붙여넣으세요.
  2. Encode로 Base64 인코딩, Decode로 디코딩합니다.
  3. 이모지·한글 등 멀티바이트 문자도 TextEncoder/TextDecoder로 정확히 처리됩니다.
  4. Copy로 결과를 복사할 수 있습니다.

Base64는 HTTP Basic Auth 헤더, Data URI(data:image/png;base64,...), 이메일 첨부(MIME), JWT 세그먼트 등에 널리 쓰입니다. 암호화가 아니라 단순 인코딩이므로 누구나 디코딩할 수 있습니다.

예시

UTF-8 라운드트립:

Encode: "안녕 🌍" → 7JWI64WVIPCfjI0=
Decode: 7JWI64WVIPCfjI0= → "안녕 🌍"

JWT 페이로드 디코딩:

eyJzdWIiOiIxMjMiLCJuYW1lIjoiQWRhIn0
→ {"sub":"123","name":"Ada"}
