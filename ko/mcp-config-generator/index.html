Claude Desktop·Cursor·VS Code용 Model Context Protocol(MCP) 서버 설정을 생성합니다. transport·command·args·env 입력 → 클라이언트별 올바른 JSON 포맷과 설치 경로를 자동 출력.

사용 방법

            
  1. 대상 클라이언트(Claude Desktop·Cursor·VS Code)를 선택하세요.
  2. 서버 이름(dict 키로 사용됩니다)을 입력합니다.
  3. transport 선택: stdio는 로컬 프로세스(가장 일반적), streamable-http·sse는 원격 서버용.
  4. stdio: command와 args 입력 / HTTP·SSE: URL 입력.
  5. 필요시 환경 변수 추가.
  6. 생성 후 표시된 경로의 설정 파일에 JSON을 복사하세요.
설정 파일 경로:
Claude Desktop — macOS: ~/Library/Application Support/Claude/claude_desktop_config.json; Windows: %APPDATA%\Claude\claude_desktop_config.json
Cursor — 전역: ~/.cursor/mcp.json; 프로젝트별: .cursor/mcp.json
VS Code — 워크스페이스: .vscode/mcp.json (또는 User Settings → MCP Servers)
