📝 ChatGPT → 블로그 변환기

🔒 Secure & Client-side

ChatGPT(또는 Claude) 대화 — 텍스트 또는 JSON export — 를 붙여넣으면 사용자/AI 턴을 분리하고, 프롬프트를 선택적으로 제거하며, AI 응답을 블로그용 Markdown으로 정리합니다.

사용 방법

  1. ChatGPT에서 공유 아이콘 → Copy 또는 Share. 또는 데이터 export로 JSON 파일을 받으세요.
  2. 대화를 왼쪽 패널에 붙여넣으세요.
  3. 옵션 선택: 사용자 프롬프트 제거는 질문을 빼고, AI 턴 병합은 모든 답변을 하나의 글로 합치며, H1 제목 추가는 제목 헤딩을 앞에 붙입니다.
  4. 변환 후 오른쪽 Markdown을 확인하세요.
  5. .md 다운로드로 로컬 저장 가능합니다.

지원 포맷: "You said:" / "ChatGPT said:" 라벨 텍스트 (한글 "사용자" / "어시스턴트"도 인식); mapping 노드가 있는 JSON export; {messages: [{role, content}]} 일반 형식. 역할 마커가 없으면 입력 전체를 하나의 AI 응답으로 처리합니다.
