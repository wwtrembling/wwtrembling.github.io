How to use

            
  1. Drag-drop or pick a PNG/JPEG/WebP image.
  2. The tool reads PNG ancillary chunks and JPEG APP1 (EXIF) segments locally.
  3. If a Stable Diffusion parameters chunk is found, the prompt and settings appear in the first section.
  4. If EXIF UserComment contains identifying text (e.g., "Midjourney", "Generated by ..."), it shows in the second section.
  5. C2PA / Content Credentials boxes are flagged when present.
What this cannot tell you: whether an image is AI-generated when metadata has been stripped (most social platforms strip metadata on upload). A "no metadata" verdict is not a clean bill of health — many genuinely AI-generated images have no embedded fingerprint after re-saving.
